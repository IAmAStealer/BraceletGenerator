Bracelet Générateur

Bienvenu sur le site du projet Bracelet Générator !

Bracelet Générateur est un logiciel de création de patrons de bracelets brésiliens. Il permet de facilement créer ses propres patrons. Il est possible d'ajouter/d'enlever des fils et des lignes à tout moment. Le logiciel propose un large choix de couleurs pour les fils. Il permet, de plus, de symétriser horizontalement le patron. Avec l'éditeur de motifs bicolores, il suffit de créer le motif et le patron sera généré automatiquement. Il est possible d'effectuer des rotations du motif ou de le symétriser. Les patrons peuvent être exportés en .png, .jpeg et .ps.

Ce projet est hébergé sur SourceForge.
